1. Работа с удаленным репозиторием

    • Клонируем из удаленного репозитория

      git clone git@github.com:trevordmiller/utility-functions.git

    • Устанавливаем удаленный репозиторий локально

      git remote add origin git@github.com:AndreyRadyshevskiy/git-test.git

    • Отправляем на сервер

      git push -u origin master

    • Отправляем коммит на сервер

      git push

    • Скачиваем с сервера

      git pull тоже самое что и git fetch && git merge
